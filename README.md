# Interstate Realty Management Website

Static website ready for GitHub + Netlify.

## Pages
- `index.html` — Home
- `about.html` — About
- `rentals.html` — Available Rentals
- `apply.html` — Apply Online
- `maintenance.html` — Maintenance Requests
- `resources.html` — Tenant Resources
- `contact.html` — Contact

## Replace placeholders
Search the files for these placeholder values and replace them:

- `YOUR_RENTREDI_APPLICATION_LINK_HERE`
- `YOUR_RENTREDI_MAINTENANCE_LINK_HERE`
- `YOUR_RENTREDI_OR_STESSA_PORTAL_LINK_HERE`
- `YOUR_DROPBOX_UPLOAD_LINK_HERE`
- `leasing@yourdomain.com`
- `(000) 000-0000`

## Upload to GitHub
1. Create a new GitHub repository.
2. Upload all files in this folder.
3. Commit changes.

## Deploy to Netlify
1. Create a Netlify account.
2. Choose **Add new site** > **Import from GitHub**.
3. Select the repository.
4. Build command: leave blank.
5. Publish directory: leave blank or use `/`.
6. Deploy.

## Connect GoDaddy domain
1. In Netlify, go to **Domain Management**.
2. Add your custom domain.
3. Follow Netlify's DNS instructions.
4. In GoDaddy, update DNS records as Netlify instructs.

## Notes
This is a simple static website. RentRedi, Stessa, and Dropbox should be linked through buttons rather than embedded directly.
